# Multicoin (MUC)

![Multicoin Logo](share/pixmaps/multicoin.png)

Multicoin (MUC) is a cryptocurrency based on Litecoin Core 0.21.4, designed for fair distribution through the Multifaucet.eu ecosystem.

## 🪙 Specifications

* **Ticker:** MUC
* **Algorithm:** Scrypt Proof-of-Work
* **Block Time:** 2.5 minutes
* **Block Reward:** 50 MUC (initial)
* **Halving:** Every 840,000 blocks (~4 years)
* **Max Supply:** 84,000,000 MUC
* **Premine:** None
* **Genesis Block:** April 20, 2026

## 💰 Distribution

Multicoin is distributed through:

* **Faucet:** [Multifaucet.eu](https://multifaucet.eu)

* **Mining:** Solo and pool mining available
